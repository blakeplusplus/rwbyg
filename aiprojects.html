Quick Contribute Read Me

A word from Headpatter:
Use this site to edit "Transcription_master.txt". Make sure to login to your google account when editing it!

REMEMBER: ACCURACY OVER GRAMMAR!

The format is "Filename|Accurate transcription".
Please refer to 15.ai's official guide for further information.
Make sure to sign yourself on 15README.txt when you contribute! [Discord, Reddit, Twitter, etc.]


If you plan on adding lines from the show, please clean them so only voices are available. Minimal to no effects should be present.
Convert them to WAV or OGG, as we need them lossless.
Add a folder called "Web Series" and upload them from there.

The Filename convention for it will be "[EpisodeNumber]_[FirstThreeWords]".
Example: V1E1_AreYouRobbing.wav

If you can, add them to "Transcription_master.txt" by adding this syntax: [Directory]/[Filename]|[Transcription]
Example: Web Series/V1E1_WellOneTeacher.wav|Well, one teacher in particular.

IMPORTANT: MAKE SURE IT'S SHORTER THAN 12 SECONDS OR THE SYSTEM WILL NOT APPROVE IT!!!
