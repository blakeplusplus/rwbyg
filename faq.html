Moral Code

i've been criticized before for having a really terrible code of morals concerning piracy, so i think it's worth it to try and define a public code of morals on piracy to help keep myself in check.

General Statement: Piracy is meant for preservation, circumventing bullshit gatekeeping and/or anti-consumer business practices.
  1. Pirating content that is offered digitally that allows the user to OWN the content without much work is bad.
    1. Max amount of effort would be using a program like youtube-dl.
    2. If the content that is offered officially is in lower quality compared to the pirate's service, it is okay to pirate.
    3. The only exception to this rule is something like Steam; any company that is "too big to fail" AND has a history of keeping titles in their marketplace.
    4. Time-gated content done not out of excessive greed OR with a short time gap (less than a month) is not okay to pirate. RWBY is a good example.
  2. Physical media, especially that which has a chance of being lost forever, MUST be ripped and shared freely without gatekeeping.
  3. Advertising/recommending to pirate something excessively, especially when you did not rip it yourself, is bad.
    1. This website circumvents this rule by not being advertised (except sometimes to a select few privately).
  4. Providing a middleman service for pirated content is okay if there exists another party who would do the same thing as you anyway.
  5. Price gouging or other anti-consumer practices warrants piracy.
  6. Free content locked behind a sign-up page warrants piracy.
  7. Content released by pirates that falls outside of this specified moral code should not be respected.
  8. Pirates who have released content do NOT deserve to place arbitrary restrictions or limitations on their products. This includes leaked summaries, BluRay rips, etc.
  9. Above all else, always recommend to users to support the original creators when doing any service relating to piracy.
